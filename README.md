# Vibe Coder

A vibe coding project managed by Hermes AI.

## What is Vibe Coding?

Vibe coding is the practice of letting AI write code while you focus on the vision. You describe what you want, the AI generates it, and you iterate together.

## How to Use This Repo

1. Tell Hermes what you want to build
2. Hermes writes the code and commits it here
3. Review the changes on GitHub
4. Iterate and improve

## Project Structure

- `src/` - Source code
- `docs/` - Documentation
- `tests/` - Tests
